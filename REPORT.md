# Отчет по заданию: переход с cookie на токены (JWT) и защита чата SignalR

## 1) Цель работы
Реализовать аутентификацию и авторизацию на основе JWT токенов в приложении ASP.NET Core, обеспечить регистрацию и вход пользователя, а также ограничить доступ к SignalR-хабу только для авторизованных клиентов.

## 2) Что реализовано
- Настроена JWT-аутентификация в ASP.NET Core.
- Добавлены API-методы регистрации и входа:
  - `POST /api/auth/register`
  - `POST /api/auth/login`
- Генерируется JWT токен после успешного входа.
- Создан защищенный SignalR-хаб `"/chatHub"` с авторизацией.
- Клиентская часть использует токен для подключения к хабу.
- Пароли хранятся в виде PBKDF2-хеша (соль + хеш), а не в открытом виде.

## 3) Ссылки на код

- JWT-конфигурация: `appsettings.json`
